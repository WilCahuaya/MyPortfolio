This assignment consists of creating SQL queries that answer the questions on movielens-1.md

To query the movielens database you will need to create it on your local environment using the movielens.sql dump file.

After cloning the repo, navigate to it and just execute the following script: 

$ createdb movielens
$ psql -d movielens < movielens.sql

The response should be something like this:

NOTICE:  table "occupations" does not exist, skipping
DROP TABLE
CREATE TABLE

DROP TABLE

COMMIT

Then you can connect using:

$ psql -d movielens
psql (13.0)
Type "help" for help.
movielens=#

MovieLens data sets were collected by the GroupLens Research Project
at the University of Minnesota.

This data set consists of:
    * 100,000 ratings (1-5) from 943 users on 1682 movies.
    * Each user has rated at least 20 movies.
  * Simple demographic info for the users (age, gender, occupation, zip)

F. Maxwell Harper and Joseph A. Konstan. 2015. The MovieLens Datasets:
History and Context. ACM Transactions on Interactive Intelligent
Systems (TiiS) 5, 4, Article 19 (December 2015), 19 pages.
