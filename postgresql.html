Pasos para instalar postgresql

PostgreSQL Cheat Sheet

Start postgresql service:

$ sudo service postgresql start

Then you can connect using:

$ psql -d db_name
psql (13.0)
Type "help" for help.
db_name=#

From here you can start writing your queries: 

movielens=# SELECT * FROM table_name;
 id |    name
----+-------------
  1 | Action
  2 | Adventure
  3 | Animation
  4 | Childrens
  5 | Comedy
  6 | Crime
  7 | Documentary
  8 | Drama
(8 rows)
